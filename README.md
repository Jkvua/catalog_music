# Catalog Music API 🎵

API desenvolvida em **Flask** para a catalogação de álbuns de músicas

# 🚀 Requisitos

- Python 3.10+ (recomendação)
- Virtualenv

# 🚀 Ativar o ambiente

- python3 -m venv .venv
- source .venv/bin/activate #linux
- .venv\Scripts\activate #windows

# 📦 Instalação de dependências

- pip install -r requirements.txt

# 🚀 Executanto aplicações

- python3 run.py
